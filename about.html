Radiant – Business analytics using R and Shiny

Radiant code

Version: 0.9.3.0, Date: 2018-4-20

Built using Shiny by Rstudio. The full source code is available on GitHub

Package dependencies

Package dependencies: car, miniCRAN, gridExtra, GPArotation, psych, wordcloud, AlgDesign, knitr, lubridate, ggplot2, pryr, shiny, magrittr, tidyr, dplyr, broom, shiny, rmarkdown, shiny, NeuralNetTools, data.tree, nnet, DT, Gmedian
